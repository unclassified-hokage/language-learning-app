# Language Learning App

> **"Makes you fluent, not just addicted."**

An open-source, AI-driven language learning app — the Duolingo alternative that actually teaches you to speak.

## Why This Exists

Duolingo has 34 million daily users. Most can't hold a basic conversation after months of use.
This app fixes that. Free. Open source. AI-first.

## What Makes It Different

| Feature | Duolingo | This App |
|---------|----------|----------|
| AI conversation practice | $30/month (DuolingoMax) | ✅ Free |
| Grammar explained in context | Hidden/minimal | ✅ AI-powered, inline |
| Intermediate+ content | Mostly beginner | ✅ Beginner → Fluent |
| Pronunciation coaching | Right/Wrong only | ✅ Phonetic feedback |
| Hearts / punishment mechanics | Yes | ❌ Never |
| Cost | Free (with ads) or $84/year | ✅ 100% Free |
| Open source | No | ✅ Yes |
| Community-built courses | No | ✅ Yes |
